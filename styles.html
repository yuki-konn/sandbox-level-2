Resizing the background image

            Resizing the background image
            background-size: 12%;
            - 1 value sets the width and height proportionately.
            - 2 values sets the width and height individually.

Repeating a background

            Repeating a background
            background-repeat: no-repeat;
            - By default, the background repeats.
          
            Repeating a background
            background-repeat: space;
            - Repeats the background such that it doesn't get cut off.
          
            Repeating a background horizontally
            background-repeat: space no-repeat;
            - 1st value: horizontal repeat (x-direction)
            - 2nd value: vertical repeat (y-direction)
            - Values are separated by a space
          
            Repeating a background vertically
            background-repeat: no-repeat repeat;
            - 1st value: horizontal repeat (x-direction)
            - 2nd value: vertical repeat (y-direction)
            - Values are separated by a space

            Background positions

RGBA

            RGBA
            color: rgba(2, 99, 2, 0.459);
            - Sets the transparency of colors
            - Red Green Blue Alpha
            - 0 alpha is invisible
            - 1 alpha is fully visible

Opacity

            Opacity
            opacity: 0.25;
            - Sets the transparency of elements
            - 0 is invisible
            - 1 is fully visible

Display: none VS Opacity: 0

            display: none; vs opacity: 0;
            display: none;
              - makes the element invisible
              - removes the element from the browser
              - the element is still in the element tree
            opacity: 0;
              - makes the element invisible
              - does NOT remove the element from the browser
              - the element still takes up space as it is still in the element tree

Specificity

          Specificity
          Styling priority
          1. inline
          2. id
          3. class
          4. element

          - Styles are applied based on their specificity or priority.
          - Styles with lower priority get overwritten.
          - !important overwrites them all.
          - !important goes by load order.
          - The !important that gets loaded last gets applied.
Example:
Article with inline style black, id red-italic, and class blue-bold.
inline > id > class > element
Article with id red-italic, and class blue-bold.
id > class > element
Article with class blue-bold.
class > element

Shorthand Property

Padding Shorthand Property

          Padding shorthand property
          padding: 5px 6px 7px 8px;
          - Padding gets appied in a clockwise direction when 4 values are specified.
            1. top 
            2. end (left)
            3. bottom
            4. start (right)

          padding: 10px 20px;
          - Providing 2 values specifies
            1. top and bottom
            2. left and right

          padding: 10px 20px 30px;
          - Providing 3 values specifies
            1. top 
            2. left and right
            3. bottom

Margin Shorthand Property

          Margin shorthand property
          margin: 5px 6px 7px 8px;
          - Margin gets appied in a clockwise direction when 4 values are specified.
            1. top 
            2. end (left)
            3. bottom
            4. start (right)

          Margin: 10px 20px;
          - Providing 2 values specifies
            1. top and bottom
            2. left and right

          Margin: 10px 20px 30px;
          - Providing 3 values specifies
            1. top 
            2. left and right
            3. bottom
